SMS Opt-In Consent

This page documents how Human Interest Group, Inc. obtains user consent to send SMS messages, specifically one-time passcodes (OTP) for account verification.

For: A2P 10DLC Verification Use case: Two-Factor Authentication (2FA)
← Back to Home Privacy Policy

How Users Opt In

Users provide their phone number during account registration on the Human Interest mobile app (iOS and Android). Before submitting their number, users are shown a clear disclosure explaining that they will receive a one-time passcode via SMS to verify their identity. Consent is explicit and required to complete registration.

1
User downloads the app
Human Interest is available on iOS (App Store) and Android (Google Play). The user opens the app and taps "Create Account."
2
User enters their phone number
On the registration screen, the user enters their mobile phone number. A consent disclosure is displayed directly below the phone number field.
3
Consent disclosure shown
Before the user taps "Send Code," the following message is displayed on screen:
Consent disclosure shown to user

"By entering your phone number, you agree to receive a one-time verification code via SMS from Human Interest. Message and data rates may apply. Reply STOP to opt out."

4
User taps "Send Code"
By tapping this button, the user explicitly consents to receive an OTP via SMS. No SMS is sent without this action.
5
User receives OTP and enters it
The user receives a single SMS containing the verification code and enters it in the app to complete registration.

Message Details

  • Message type: Transactional only — one-time passcodes (OTP) for account verification
  • Sender: Human Interest Group, Inc.
  • Frequency: Only upon user request (account registration, login, or account recovery)
  • No marketing messages are sent via SMS
Sample message sent to user:

"Your Human Interest verification code is 847291. This code expires in 10 minutes. Do not share it with anyone. Reply STOP to opt out."

Opt-Out Instructions

Users can opt out of SMS messages at any time by replying STOP to any message. For help, users can reply HELP or contact support@humaninterestapp.com.

No mobile information will be shared with third parties or affiliates for marketing or promotional purposes. All other categories exclude text messaging originator opt-in data and consent; this information will not be shared with any third parties.
